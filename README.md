# libasm

An assembly project for 42.

Function reimplemented in assembly :
- strlen (as ft_strlen)
- strcpy (as ft_strcpy)
- strcmp (as ft_strcmp)
- write (as ft_write)
- read (as ft_read)
- strdup (as ft_strdup)

Features :
- Syscall errors must be managed.
- errno must be updated.

Usage :
- `make` : compile the `libasm.a` library.
- `make test` : compile a `test` program.
- `./test` : execute the tests.
